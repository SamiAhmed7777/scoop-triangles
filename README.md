# Scoop Bucket for Cryptographic Triangles

## Install

```powershell
scoop bucket add triangles https://github.com/SamiAhmed7777/scoop-triangles
scoop install triangles
```

## Update

```powershell
scoop update triangles
```

## What is Cryptographic Triangles?

Privacy-focused cryptocurrency featuring Proof-of-Stake consensus, Tor v3 onion routing, and built-in encrypted messaging.

- Website: https://cryptographic-triangles.org
- Explorer: https://blocks.cryptographic-triangles.org
- GitHub: https://github.com/SamiAhmed7777/triangles_v5
